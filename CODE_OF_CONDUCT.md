# Code of Conduct

## Our Standards

**Be respectful. Be constructive. Build together.**

## Expected Behavior

- **Be respectful** - Treat everyone with courtesy and consideration
- **Be constructive** - Provide helpful feedback and criticism
- **Be collaborative** - Help newcomers and share knowledge
- **Be professional** - Maintain a welcoming environment for all

## Unacceptable Behavior

- Harassment or discriminatory language
- Personal attacks or insults
- Disruptive or inappropriate content
- Excessive self-promotion or spam

## Enforcement

If community standards are not met:

1. **Warning** - Private request to modify behavior
2. **Temporary suspension** - If behavior continues
3. **Permanent ban** - For severe or repeated violations

## Reporting

To report issues:
- Open a confidential issue in the .github repository
- DM the maintainers directly on GitHub

All reports will be handled professionally and confidentially.
